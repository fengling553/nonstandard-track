🛠️ 项目化学习

用真实项目替代刷题，在解决问题中积累可展示的能力成果

💡 做中学的方法论
用真实项目替代刷题。选一个你感兴趣的项目→拆解需要什么学科知识→缺什么补什么→完成项目→更新能力图谱。做中学，比做100张卷子管用。
智能浇花器
编程 物理(电路) 生物(植物) 数据分析
用土壤湿度传感器+Arduino+水泵，做自动浇花系统
⏱ 预计时长：1-4周 💰 费用：套件约50-150元
🛠 需要工具
Arduino 土壤湿度传感器 水泵模块
L1
用现成套件拼装，理解传感器基本概念
L2
自己写简单逻辑：湿度低于阈值就浇水
L3
加入手机远程控制和数据记录
L4
多传感器联动+机器学习预测浇水时间
L5
设计完整智慧农业系统，论文级项目
AI垃圾分类器
编程 AI/机器学习 工程 环保
用摄像头+AI模型识别垃圾类型并自动分类
⏱ 预计时长：2-8周 💰 费用：硬件约100-300元
🛠 需要工具
摄像头 树莓派/Arduino TensorFlow Lite
L1
用现成APP拍照识别，理解AI分类概念
L2
用Scratch/Python调用现成API做分类器
L3
训练自己的分类模型，提高识别准确率
L4
加上机械臂自动分拣，软硬结合
L5
优化模型到可部署水平，参加科创竞赛
个人气象站
编程 物理(热学) 数学(统计) 数据可视化
搭建气象数据采集站，长期记录分析天气数据
⏱ 预计时长：持续项目，2周搭建+长期运行 💰 费用：约80-200元
🛠 需要工具
温湿度传感器 Arduino/树莓派 Python
L1
用温度计和本子记录，学会观察和数据记录
L2
用传感器自动采集温湿度，存储到电脑
L3
搭建Web页面实时展示数据，学数据可视化
L4
用历史数据做天气预报模型
L5
与专业气象数据对比验证，写成研究报告
个人公众号/博客
写作 逻辑思维 读者心理 排版设计
从0开始运营一个内容账号，选题→写作→排版→发布
⏱ 预计时长：持续项目，至少3个月 💰 费用：免费
🛠 需要工具
微信公众号/Notion/博客平台 Markdown编辑器
L1
每周写一篇短文，培养输出习惯
L2
学会选题和结构化写作，文章有清晰逻辑
L3
研究读者需求，做选题规划，持续稳定更新
L4
形成个人风格，内容有独到见解，吸引固定读者
L5
内容有影响力，获得行业认可或商业合作
AI辅助播客
写作 口语表达 音频制作 内容策划
用AI工具制作播客节目，从选题到发布全流程
⏱ 预计时长：1-2周做出第一期 💰 费用：免费起步
🛠 需要工具
AI播客生成工具 录音设备 音频编辑软件
L1
用AI生成播客脚本，录一段3分钟音频
L2
自己写脚本+AI辅助润色，学会音频剪辑
L3
做系列播客，每期一个主题，持续输出
L4
加入采访、讨论环节，提升内容深度
L5
播客有稳定听众，内容有行业影响力
纪录片/短视频
叙事能力 视觉表达 剪辑 采访沟通
拍摄制作一条3-5分钟的短片，讲一个真实的故事
⏱ 预计时长：1-4周 💰 费用：免费（手机拍摄）
🛠 需要工具
手机/相机 剪映/CapCut AI辅助脚本
L1
用手机拍一段vlog，学会基本拍摄和剪辑
L2
做一条有主题的短视频，有开头中间结尾
L3
做一条迷你纪录片，有采访、有叙事、有观点
L4
系统化创作系列内容，形成个人风格
L5
作品参加青少年影像节/竞赛
社区调研报告
社会观察 问卷设计 数据分析 报告写作
选择一个社区问题，做调研、分析、提出解决方案
⏱ 预计时长：2-6周 💰 费用：免费
🛠 需要工具
问卷星/腾讯问卷 Excel/Python Word/飞书文档
L1

L2
设计结构化问卷，采访20人，用图表展示数据
L3
做完整的调研报告，有问题分析+数据支撑+解决方案
L4
调研报告有学术规范，引用文献，结论有创新性
L5
报告达到可发表水平，参加社科类竞赛

用AI工具辅助研究一个学科问题，产出研究论文

Semantic Scholar/Elicit Python LaTeX/Word
L1

L2

L3
用AI分析数据/做实验，写有方法论的论文
L4

L5
